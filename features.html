기능

BooQR이 제공하는 핵심 기능을 8가지로 정리했습니다.

1. QR 발급 — 서명된 JWT 기반

예약이 확정되면 Ed25519 서명된 JWT를 QR 코드에 담아 발급합니다. QR 자체에 위변조 방지 서명이 들어 있어 통신 없이도 검증할 수 있습니다.

2. 다중 수집 채널

기존 통합예약 시스템과 연결하는 방식을 3가지 중 선택할 수 있습니다. DB 트리거 / 백엔드 알림 후킹 / 프론트엔드 JS 스니펫. 기관별 사정에 맞춰 도입 부담을 최소화합니다.

3. 다중 발송 채널

SMS, 카카오 알림톡, 이메일, 웹 포털 lookup을 모두 지원합니다. 기관별로 우선순위·폴백을 자유롭게 설정할 수 있어 알림톡 실패 시 자동으로 SMS로 전환되는 폴백 처리가 가능합니다.

4. PWA 모바일 스캐너

관리자가 별도 앱을 설치하지 않고 모바일 브라우저에서 홈 화면에 추가만 하면 사용 가능. 첫 사용 시 카메라 권한 1회 허용 후 그대로 운영에 투입할 수 있습니다.

5. 오프라인 검증

입장 시작 30분 전 사전 동기화로 취소 명단·공개키를 미리 다운로드. 이후 통신이 끊겨도 IndexedDB의 데이터로 검증을 수행하고, 체크인 결과는 펜딩 큐에 쌓아 두었다가 복구 시 자동 동기화합니다.

6. 운영 실적 자동 집계

예약 수·입장 수·입장률·노쇼율·시간대 분포·평균 처리 시간·직원별 처리 건수가 자동으로 모입니다. 일·주·월 단위로 집계하며 CSV 내보내기를 제공합니다.

7. 광고/홍보 모듈

방문자의 QR 티켓 페이지 하단에 같은 기관의 다른 프로그램을 노출. 노출/클릭 통계로 광고 효과를 측정할 수 있습니다. 외부 광고주·정산은 본 시스템 범위 밖입니다.

8. 개인정보 LCM

기본 보존기간 90일(기관별 설정 가능) 후 자동 비식별화, 즉시 삭제 기능, 모든 조회·내보내기·삭제 행위에 대한 감사 로그. 개인정보보호법상 위탁 처리자 위치에서 동작합니다.
